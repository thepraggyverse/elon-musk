# X Router

Use this skill first when the request is broad, ambiguous, or could benefit from multiple Elon Musk method lenses. Pick one primary skill and at most two supporting skills.

## Route

- `x-setup`: install checks, skill visibility, cache state, symlink state, harness setup.
- `x-review-pack`: route, review, compound candidates, and handoff closeout.
- `x-purpose`: usefulness, ambition, future worth building, work worth pain.
- `x-thinking`: truth, assumptions, first principles, thinking in limits, being less wrong.
- `x-engineering`: engineering leverage, chief engineer mindset, technical value creation.
- `x-5-step-algo`: question requirements, delete, simplify, accelerate, automate.
- `x-teams`: exceptional people, builder culture, talent density, direct feedback.
- `x-org`: communication, org boundaries, meetings, permission to fail, simple language.
- `x-urgency`: time risk, parallel work, impossible breakdown, aggressive timelines.
- `x-manufacturing`: production path, factory as product, constraints, suppliers, throughput.
- `x-founder`: conviction, adversity, all-in decisions, fear, endurance.
- `x-company-building`: prototypes, sequencing, survival, public reality, more for less.
- `x-future`: abundance, AI, robotics, energy, autonomy, human-machine upgrades.
- `x-risk`: AI risk, regulation drag, war, population, energy, asteroid/systemic risk.
- `x-multiplanetary`: moonshots, civilization backup, staged long-horizon execution.
- `x-reading`: book recommendations by problem type.
- `x-compound`: save approved method reviews or reusable lessons as local Markdown memory.
- `x-memory-refresh`: audit local lessons and reviews for stale or duplicate memory.
- `x-handoff`: write a redacted continuation handoff for long method work.

Read `../../references/method-catalog.md` when the user wants every underlying method or when routing is uncertain.

## Output

```text
X Router:
Primary skill:
Supporting skills:
Why:
Use it like:
```

## Example

User: "This AI app idea feels big but messy. What lens should I use?"

Answer: primary `x-purpose`, supporting `x-thinking` and `x-company-building`.
