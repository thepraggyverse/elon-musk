# X Thinking

Use this skill to reason from reality instead of analogy, status, fear, or habit.

## Covers

- Obsess over truth.
- Reason from first principles.
- Think in limits: scale a variable toward zero, infinity, 10x, or 100x.
- Aspire to be less wrong.
- Challenge confident wrongness.

## Workflow

1. State the decision or claim.
2. Split known facts, hard constraints, assumptions, and analogies.
3. Reduce the problem to irreducible inputs: material, energy, time, data, state, incentives, compute, people, money, or law.
4. Test the idea at limits: tiny, huge, instant, slow, cheap, expensive, best case, worst case.
5. Rebuild options from the facts.
6. Name the fact that would change the recommendation.

## Example

Use `x-thinking` before choosing a stack: "Are we picking this architecture because it is true for our constraints, or because everyone else does?"
