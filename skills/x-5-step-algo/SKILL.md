# X 5 Step Algo

Use this skill before optimizing or automating. The order matters.

## Covers

- Question every requirement.
- Delete the part or process.
- Simplify and optimize what remains.
- Accelerate cycle time.
- Automate last.
- Overdelete and define add-back triggers.

## Workflow

1. Question requirements: name the owner, reason, evidence, and current validity.
2. Delete: remove features, steps, fields, meetings, approvals, code paths, or tools that do not prove necessity.
3. Simplify: make the remaining path boring, direct, reversible, and inspectable.
4. Accelerate: shorten the loop only after the direction is valid.
5. Automate: automate stable repeated work with visible failure modes.

## Example

Use `x-5-step-algo` before building a dashboard: "Which requirements are real, what can be deleted, and what should not be automated yet?"
