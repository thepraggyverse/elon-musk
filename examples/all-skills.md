and the expected output artifact.

## Close The Loop Example

```text
Use x-router on this launch plan.
Then apply the primary method.
Then use x-compound to save the reusable lesson.
Then use x-handoff if the next session needs to continue the work.
```

Expected result: a review that produces a decision, a small reusable note when
approved, and a compact continuation artifact when needed.
