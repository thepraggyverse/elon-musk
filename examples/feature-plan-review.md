# Feature Plan Review Example

Prompt:

```text
Use x-5-step-algo on this feature plan:
We want a team dashboard with daily KPIs, Slack alerts, CSV export, role-based permissions, and scheduled email digests.
```

Expected focus:

- Which requirements have real owners?
- Which features can be deleted or delayed?
- What is the smallest dashboard that proves value?
- What should not be automated yet?
