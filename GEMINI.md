# Gemini Notes

This repository publishes the `elon-musk` portable skill pack. Use
`skills/x-*/SKILL.md` as the source of truth and `docs/SKILL_INDEX.md` as the
generated skill index.

Good starting prompts:

```text
Use x-setup to check whether this plugin is installed and visible.
Use x-review-pack to review this plan and close the loop.
Use x-router to choose the right Elon Musk method lens.
```

Keep memory local and approved: `x-compound` and `x-memory-refresh` do not read
raw session logs by default.
