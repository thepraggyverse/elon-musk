# Source Boundaries

This repo is derived from the user's method extraction from *The Book of Elon*, but it is not a copy of the book.

## What This Repo Contains

- Original skill workflows.
- Paraphrased method summaries.
- Section-level coverage maps.
- Install scripts, validation scripts, examples, and docs.

## What This Repo Does Not Contain

- Full text of *The Book of Elon*.
- Long quotes from the book.
- EPUB files or extracted chapters.
- Transcripts, videos, copyrighted commentary, or private notes.

## Attribution

Use section-level descriptions, not long excerpts.
Use `docs/SOURCES.md` to distinguish direct, near-primary, book-derived, and derived material.

Acceptable:

```text
This method maps to the book's "The Algorithm" and "Designing the Organization" sections.
```

Avoid:

```text
Copying paragraphs from the book into a skill or README.
```

## Public Repo Rule

If a future change needs exact source quotes, keep them short, cite the source, and prefer links or paraphrases. Do not commit source documents, generated full-text extracts, or private reading notes.
