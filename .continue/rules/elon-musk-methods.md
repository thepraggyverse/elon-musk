# Elon Musk Methods

This repo exposes portable `SKILL.md` folders under `skills/`.

Use `x-review-pack` for full close-the-loop reviews, `x-router` for choosing a
method lens, `x-setup` for install diagnostics, `x-compound` for approved local
lessons, `x-memory-refresh` for stale lesson cleanup, and `x-handoff` for
continuation notes.

Follow the selected skill's output shape and write files only when that skill
allows it.
