# OpenCode Notes

This repository is a portable skill pack. The canonical instructions live in
`AGENTS.md`; the callable skill folders live under `skills/`.

Recommended entry points:

- `x-setup` for install diagnostics.
- `x-review-pack` for a full close-the-loop review.
- `x-router` for choosing a method lens.
- `x-compound` for approved local Markdown lessons.
- `x-memory-refresh` for stale or duplicate saved lessons.
- `x-handoff` for redacted continuation notes.

Use direct `SKILL.md` folders when OpenCode does not support this repository as
a native marketplace package.
